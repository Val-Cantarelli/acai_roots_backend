Schedule Management

Basic timetable organization developed using Java (Spring Boot).

About

This project involves creating a website that seeks to streamline the service scheduling process, guaranteeing dependability while staying within the financial means of small businesses. To achieve this, the website leverages a cost-effective cloud service (fly.io), coupled with well-established technology like the Java language and REST architecture.

Getting Started

To run this project, install it locally using mvn:

 mvn <colocar comando>

By Docker:

 docker build .

By Fly.io: - Create an account at: https://fly.io/ - Config the CLI

fly launch
